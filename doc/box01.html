box의 기본 성질 (display:block)

  • 투명함
  • width,height이 auto
  • Y축으로 정렬됨 / 따라서 블록요소 옆에 블록이 올수 없음
  1. 태그는 크게 inline / block 으로 나뉨
  2. 글자 그림 등 내용에 관여하는 태그 span strong em icon input 은 인란라인 속성을 가지고 나머지는 block속성을 가진다.
  3. block 속성은 width height margin padding 값을 갖고 inline속성은 좌우 margin padding 만을 가진다
  4. inline 상하 padding의 경우 background를 포함하여 padding이 적용된 것 처럼 보인다.

  • javascript의 적용 범위는 인타일 스타일이므로 외부 내부보다 우선시됨
